# webpack

## 1. Installation
``` bash
# setp1. install webpack and bower
$ npm install webpack -g
$ npm install bower -g
  
# setp2. install project packages
$ cd ./MySample/webpack
$ npm install
$ bower install
```

## 2. Build Module
``` bash
$ grunt build-dev
```
